Getting EyeDB sources

The EyeDB sources can be downloaded from project page on sourceforge (http://sourceforge.net/projects/eyedb). By following the "download" link, you will get to the list of releases available for downloads. Source archives are provided under .tar.bz2 and .tar.gz format.

After downloading the source archive, unpack it using tar. This will create a directory named eyedb-VERSION where VERSION is the version number of EyeDB, for example eyedb-2.8.7.
